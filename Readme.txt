Solitaire Game Code VS C++

----------------------------------------------

Follow the steps for executing
	1. Open Visual Studio

	2. Create a console application Program from template menu

	3. Open Solution Explorer, and delete .cpp file from source menu

	4.  Add All these files by 
		i ) either select all files and dragg to source folder in solution explorer
		ii) or Right click on source folder > Add exixting file > locate the files > select > add

	5. In MENU bar select project > properties > linker > Input > additional dependencies > edit and add following in the additional dependencies and then select ok
		Winmm.lib

	6. inside linker open System > subsystem > select console. 
	 
	7.. Build and run your program, Happy Execution :)
